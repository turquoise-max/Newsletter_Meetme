# AI Newsletter SaaS 프로젝트 기능 및 혁신 아이디어 리포트

## 1. 프로젝트 개요
본 프로젝트는 사용자가 입력한 주제를 바탕으로 최신 정보를 수집하고, AI 수석 편집장의 시각으로 고품질의 뉴스레터를 자동 생성하며, 이를 미려한 디자인의 HTML로 배포할 수 있는 통합 SaaS 솔루션입니다.

---

## 2. 핵심 구현 기능

### A. 고도화된 자료 수집 엔진 (Smart Crawler)
*   **멀티 쿼리 확장:** 입력된 주제를 기술, 비즈니스, 사례 연구 등 3가지 심화 관점으로 자동 확장하여 검색의 깊이를 확보합니다.
*   **실시간 유효성 검증:** 수집된 모든 URL과 이미지에 대해 HEAD/GET 요청을 시도하여 404 에러나 깨진 링크를 실시간으로 필터링합니다.
*   **메타데이터 추출:** 아티클의 실제 발행일(published_date)을 수집하여 본문 서술 시 최신성을 반영합니다.

### B. AI 수석 편집장 엔진 (Intelligence Generator)
*   **'밑미(meet me)' 스타일 스토리텔링:** 정보를 단순히 나열하지 않고, 독자에게 공감과 질문을 던지는 에세이스트형 문체를 구사합니다.
*   **개별 아티클 1:1 블록 매칭:** 전체 요약 방식의 오류를 해결하기 위해 수집된 각 소스를 독립된 블록으로 변환하고 실제 URL을 강제 바인딩(Injection)하여 링크 신뢰도를 100% 확보합니다.
*   **지능형 이미지 매핑:** 수집된 이미지 중 문맥에 가장 적합한 이미지를 AI가 스스로 판단하여 적재적소에 배치합니다.

### C. 프리미엄 블록형 에디터 (Tiptap 기반)
*   **고도화된 인터랙션:** Tiptap 라이브러리를 통해 정교한 텍스트 편집 및 드래그 앤 드롭 기능을 제공합니다.
*   **3단 레이아웃:** [자료 및 이미지] - [에디터] - [템플릿]의 직관적인 워크플로우를 통해 생산성을 극대화합니다.
*   **단계별 진행바:** 생성 과정을 6단계(수집, 분석, 테마 설정 등)로 세분화하여 시각적으로 진행 상황을 사용자에게 알립니다.

## 3. 적용된 혁신적 사고 및 아이디어

### 💡 정보의 혈통(Link of Trust) 복원
*   **아이디어:** "URL은 에디터의 명예다."
*   **구현:** AI가 가짜 링크를 생성하는 할루시네이션을 방지하기 위해, 백엔드 최종 단계에서 수집된 원본 소스 목록과 생성된 블록의 URL을 대조하여 강제로 일치시키는 **URL Injection 포스트 프로세서**를 적용했습니다.

### 💡 시간적 앵커링 (Temporal Anchoring)
*   **아이디어:** AI의 학습 데이터(과거)와 검색 결과(현재)의 시점 차이 극복.
*   **구현:** 시스템 프롬프트에 현재 날짜(2026-01-27)를 고정 주입하고, 모든 아티클에 발행일 태그를 붙여 **"어제 발표된...", "지난주에 화제가 된..."**과 같은 실제 시점 기반 서술을 가능하게 했습니다.

### 💡 편집장-기자 듀얼 시스템 (Agentic Workflow)
*   **아이디어:** 역할 분리를 통한 정보 왜곡 차단.
*   **구현:** `Crawler`는 사실과 데이터 확보에 집중하고, `Generator`는 수집된 데이터를 바탕으로 전체 흐름과 브릿지(연결 문구)를 만드는 데 집중하는 구조를 채택했습니다.

### 💡 디자인 일관성 (Visual Fidelity)
*   **아이디어:** "에디터에서 보던 감동을 이메일 클라이언트에서도 그대로."
*   **구현:** Tailwind CSS v4와 Radix UI의 모던한 스타일을 HTML 내보내기 로직(`html-exporter.ts`)에 100% 인라인 CSS로 이식하여, 다운로드한 HTML의 시각적 품질을 보장합니다.
